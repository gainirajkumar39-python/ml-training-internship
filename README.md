# ml-training-internship
**Dataset:**
I am choosing the Medicare Provider Utilization and Payment dataset. It contains around 50+ columns and is available in CSV/Excel format. The dataset provides detailed information about healthcare providers, including hospital details, patient demographics, medical conditions, treatment statistics, and payment-related data.

**Problem Statement:**
Build a predictive model to estimate healthcare risk scores and patient condition severity based on provider-level and patient-level historical data.
**Title:**
“HEALTHCARE RISK PREDICTION AND ANALYSIS”

**Objective:**
The main goal of this project is to analyze healthcare data and develop a predictive model that can estimate patient risk scores and identify patterns in medical conditions across different hospitals and regions.

**Columns impacting project:**
Provider Details: Rndrng_Prvdr_Org_Name, Rndrng_Prvdr_City, Rndrng_Prvdr_State_Abrvtn
Patient Demographics: Bene_Avg_Age, Bene_Feml_Cnt, Bene_Male_Cnt, Bene_Race_Wht_Cnt, Bene_Race_Black_Cnt, Bene_Race_Hspnc_Cnt
Healthcare Usage: Tot_Benes, Tot_Dschrgs, Tot_Cvrd_Days
Financial Information: Tot_Submtd_Cvrd_Chrg_Amt, Tot_Pymt_Amt, Tot_Mdcr_Pymt_Amt

**Conclusion:**
This dataset helps in understanding healthcare patterns, patient demographics, and disease prevalence. The predictive insights can support better decision-making in healthcare management, improve patient outcomes, and optimize resource utilization.
